# Srashti Gupta — Software Engineering Portfolio

A responsive, recruiter-focused portfolio for software engineering internship roles.

## Highlights

- Featured Alpha Sign hackathon case study with live product and repository links
- Impact-first experience bullets backed by résumé metrics
- Responsive project, experience, education, skills, testimonial, and contact sections
- Accessible navigation, reduced-motion support, semantic markup, and validated HTML
- Lightweight static implementation with no build step or framework dependency

## Run locally

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

## Deploy

The site is ready for GitHub Pages. Push the files on the `main` branch and keep Pages configured to deploy from the repository root.
