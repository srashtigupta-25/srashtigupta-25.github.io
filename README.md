# Srashti Gupta — Software Engineering Portfolio

A light, interactive portfolio focused on production engineering experience, measurable impact, and selected systems, cloud, AI, and data projects.

## Technology

- React 19
- Vite
- Responsive CSS
- Canvas-based network animation
- Accessible keyboard command palette

The production build is deployed from the repository root through GitHub Pages. Editable source is stored in [`app/`](app/).

## Run locally

```bash
cd app
npm install
npm run dev
```

## Build

```bash
cd app
npm run build
```

Copy the resulting `app/dist/` files to the repository root before publishing.
